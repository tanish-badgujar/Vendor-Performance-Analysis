# 📊 Vendor Performance Analysis

## 📌 Project Overview
This project focuses on analyzing vendor performance, profitability, purchasing behavior, and inventory risks using Exploratory Data Analysis (EDA) and statistical analysis techniques.

The objective was to identify:
- High-performing and low-performing vendors
- Profitability trends
- Inventory risks
- Vendor dependency
- Business optimization opportunities

## 📊 Key Business Insights

### ✅ Vendor Concentration Risk
- Top 10 vendors contribute **65.69%** of total purchases
- Heavy dependency on limited vendors creates supply chain risk

### ✅ Inventory Risk
- Identified **$2.71M** tied up in slow-moving inventory
- Excess stock impacts storage cost and cash flow

### ✅ High Margin Opportunity
- Found **198 brands** with high profit margins but low sales
- These products can benefit from targeted marketing strategies

### ✅ Bulk Purchasing Impact
- Large purchase orders significantly reduced unit costs
- Bulk buying improved purchasing efficiency

### ✅ Statistical Analysis
- Hypothesis testing revealed a significant difference in vendor profitability models
- Null hypothesis was rejected based on p-value analysis
